# Hi3Helper.Plugin.Endfield.CN

Collapse启动器的第三方插件，以支持明日方舟：终末地的下载更新启动

当前暂时仅支持识别并检测是否有更新，暂时不支持任何更新操作和安装操作

<img width="1278" height="711" alt="image" src="https://github.com/user-attachments/assets/f3f572c0-bfb7-4436-b8e7-47765f42c052" />

## 当前能够正常工作的功能如下：
- [x] 检测是否为最新版本
- [x] 获取背景图、banner、新闻
- [x] 启动游戏/检测游戏
- [ ] 社媒面板(查看插件对应数据结构中)

## 预计功能/未编写完成功能：
- [ ] 更新（当前无1.0.0版本无法进行测试）
- [ ] 下载游戏
- [ ] 预下载（等待官方启动器预下载）
- [ ] 检查游戏完整性（启动器似乎暂时没有此功能的对应接口）
- [ ] 对接多个服务器（如全球服，b服）
